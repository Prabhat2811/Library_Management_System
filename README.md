# 📚 Library Management System

A backend-based **Library Management System** developed using **Spring Boot**, **Spring Data JPA**, and **RESTful APIs**. This project is designed to perform basic **CRUD operations** for managing library books and records efficiently.

This is a backend-only application and does **not** include any frontend UI, JSP pages, or view layer. All API requests and responses are tested using **Postman**.

---

## 📌 Project Overview

The main objective of this project is to manage book records using REST APIs. It allows users to add, fetch, update, and delete books from the database through HTTP requests.

The application runs on the **embedded Apache Tomcat Server** provided by Spring Boot, so no external server setup is required.
